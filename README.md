Micro Hack , challange 1
This is micor challenge 1 , the challenge is a simple , given a list of connected nodes , find the node(s) that have no direct or nondirect connection to the number 3 node .
## Example :

6 1                              
0 5                            
4 5                           
2 0                              
2 3                             
4 2

8 4
